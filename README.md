# What's in My Closet!

### 프로젝트 소개

다음과 같은 분들을 위한 옷장 관리 웹 페이지입니다.
* 내 옷장에 어떤 옷이 있는지 기억나지 않아 매번 같은 옷을 구매하시는 분들
* 어떤 옷을 입을까 고민하시는 분들

옷의 종류, 스타일, 색상, 계절을 사진과 함께 저장할 수 있습니다. 등록된 옷의 전체 목록을 확인할 수 있으며, 카테고리를 통해 혹은 검색을 통해 특정 옷만 확인할 수도 있습니다. 또한, 기온을 입력하면 알맞은 옷을 찾아 추천해 드립니다!


### 포함된 프레임워크/라이브러리

* Bootstrap
* JQuery
* Flask
* Pymongo
